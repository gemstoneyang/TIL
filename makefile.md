# MAKEFILE

## %.o : %.c

- % 자리에는 무엇이든 들어갈 수 있음
- 앞에 %에 들어온 것이 뒤에 의존성 부분에도 같이 들어감

## 단축어

- @ : 만들어야 할 타겟
- < : 의존성 중 첫 번째 요소
- ^ : 의존성 전체 요소

## .PHONY

- 특별 타겟
- 원래 타겟은 그 타겟을 생성한다고 생각하고 실행됨
- .PHONY에 등록된 타겟은 실제로 그 타겟을 생성하지 않고 명령어로만 실행하겠다는 것
    
    = 가짜 타겟
    

## make -j 옵션

- 병렬 실행
- 컴파일 속도가 향상됨
- ex) make -j 8 : 8개 프로세스에서 병렬 처리

## submake 문제 해결

- make 명령어가 또 다른 makefile을 실행하는 것은 문제가 있음
- makefile 하나로 모든 컴파일을 수행할 수 있도록 해야됨

## cc와 플래그

- cc : POSIX 표준으로 정의됨
- -Wall -Wextra -Werror : gcc에 정의되어 있음
